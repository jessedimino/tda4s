# tda4s - TDA For Scala

This project is intended to be an accessible development platform for new and 
experienced developers of TDA methodology.
It aims to provide both baseline reference and optimized algorithms for the 
most common procedures in TDA, and to be easy to use both for newcomers to TDA
and experienced analysts.

The project builds on the heritage from PLEX, jPlex, javaPlex, with their 
origins in the Stanford group centered around Gunnar Carlsson, with previous
contributions from Vin de Silva, Harlan Sexton, Andrew Tausz, Henry Adams,
and Mikael Vejdemo-Johansson.
